Query: change directory
Response: The primary command to change directories in UNIX is `cd`.

*   **`cd`**: Followed by the directory path, it changes the current working directory. For example, `cd /home/user/documents` will change the directory to `/home/user/documents`.  If used without any path, `cd` will take you to your home directory.

While not directly related to *changing* directories, `pwd` is useful for *knowing* the current directory:

*   **`pwd`**: Prints the absolute path of the current working directory.
